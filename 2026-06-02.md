## 2. MELD: Mel-Spectrogram-Based Speech Language Modeling
📄 *2026年5月28日（最新）*

- **联合优化**编码器 + 自回归模型，支持 TTS + STT
- 引入离散潜变量模型，解决传统两阶段训练的编码器不感知下游任务问题
- 有效抑制 mel 频谱图生成中的静音卡死和丢词
- 单一模型同时完成 TTS、STT、TTS-STT
- 论文链接：https://arxiv.org/html/2605.29859v1

**关键结论：**
- 联合优化在 STT 上比 Codec 方案 WER 降低 6%
- 比 dMel 在 test-other 上提升 0.4-3.2%
- 多任务训练（TTS+STT）缩小了与单任务模型的差距

## 3. TASTE: Text-Aligned Speech Tokenization and Embedding
📄 *ICLR 2026*

- 提出**语音 token 与文本 token 对齐**的新方法
- 基于注意力机制的聚合，通过语音重建学习联合分词
- 大幅缩短 token 序列长度，减少模态差异
- 基于 LoRA 在预训练 LLM 上做联合语音建模
- **1.3B 参数超越同类 7B 预训练 SLM**

论文链接：https://arxiv.org/pdf/2504.07053

---

## 4. SiTok: Scaling Speech Tokenizers with Diffusion Autoencoders
📄 *ICLR 2026*

- 1.6B 参数，200 万小时训练数据
- **12.5Hz 极低 token 率 + 200bps** 实现高保真重建
- 扩散自编码器联合优化量化与重建
- 引入 CTC 语义正则化，统一支持语音理解与生成
- Shortcut Fine-tuning 加速扩散解码

论文链接：https://arxiv.org/pdf/2602.06602

**性能亮点：**
- 语音理解（情感识别、关键词检测、说话人验证、ASR）全面超越基线
- 零样本 TTS 质量优异
